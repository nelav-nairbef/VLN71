Recent Projects

Project 1

Internet of Things (Iot)

"Sistem Presensi Elektronik Terintegrasi RFID untuk Akademik (SPEKTRA 1.0)" adalah proyek berbasis IoT yang dirancang untuk mencatat presensi secara real-time menggunakan teknologi RFID. Dikembangkan dengan dukungan MSIB Indobot Academy dan Kampus Merdeka RI, sistem ini menawarkan solusi presensi yang efisien dan akurat untuk mendukung digitalisasi akademik.

Project 2

Robotika

Proyek ini adalah robot line follower yang dirancang untuk lomba antar SMA se-Bengkulu. Robot ini mengikuti jalur dengan presisi tinggi menggunakan sensor canggih dan algoritma efisien, mengutamakan kecepatan, stabilitas, dan akurasi di jalur kompleks. Proyek ini juga bertujuan untuk mendorong minat siswa dalam bidang teknologi robotika.

Project 3

Desainer grafis

Sebagai desainer grafis, saya menciptakan desain visual untuk dipublikasikan di media sosial. Proyek ini fokus pada pembuatan materi yang menarik dan sesuai dengan identitas merek, untuk meningkatkan daya tarik visual dan interaksi dengan audiens.

Project 4

Web Portofolio

Proyek web biodata ini saya buat untuk mengembangkan keterampilan baru dan memperkenalkan diri dengan cara yang menarik. Melalui proyek ini, saya berharap bisa terus belajar dan meningkatkan kemampuan saya.

Project 5

Merancang instalasi listrik dan denah rumah di AutoCAD

Di proyek ini, saya merancang instalasi listrik dan denah rumah menggunakan AutoCAD. Saya membuat Single Line Diagram (SLD), denah ruangan, pemipaan, dan wiring untuk memastikan instalasi listrik di rumah bisa berjalan dengan aman dan efisien. Semua perencanaan ini dibuat dengan detail agar sesuai dengan kebutuhan dan kenyamanan penghuninya.
